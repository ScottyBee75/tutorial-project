How to Create a Lightbox on a Webpage

  1. What is a Lightbox?

    A Lightbox is a technique using HTML/CSS/JavaScript for enlarging an image on a webpage using an overlay. This overlay allows the user to still see the original "thumbnail" behind the expanded image. This creates the illusion to the user that they are still on the same webpage and eliminates the need for excessive back and forward navigation to view other images.
